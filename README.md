Gym Management Database System
Overview
This project is a comprehensive Gym Management System designed to streamline daily operations for fitness centers. It bridges the gap between backend data storage and application logic, providing an efficient, structured approach to managing member records, tracking trainer schedules, and processing financial transactions.

Key Features
Member Management: Secure creation, tracking, and updating of member profiles and subscription tiers.

Staff & Trainer Allocation: Organized tracking of employee shifts and the assignment of personal trainers to specific members.

Billing & Transactions: Financial record keeping for membership renewals and payment tracking.

Relational Architecture: A fully normalized database design ensuring data integrity, optimized relationships, and efficient query execution.

Tech Stack
Database: SQL / SQL Server Management Studio

Application Logic: Python / C++
